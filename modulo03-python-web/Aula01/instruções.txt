
Instalação do Django

Criação do projeto Django
    django-admin startproject <nome-do-projeto>

Criação do pacote dentro do Django
    python manage.py startapp <nome-do-projeto>

Como o Django retorna uma função view
    1. Recebe a URL informada pelo usuário
    2. Verificar se essa URL está cadastrada no arquivo urls.py do projetor
    3. Caso esteja cadastrada o Django carrega o módulo urls do pacote (urls.py)
    4. Verifica se a URL existe dentro do módulo, se existir chama a função associada a rota
